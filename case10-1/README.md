Туризм в регионах РФ

Работа с базой данных по туризму по регионам России

задача - изучить распределение количества гостиниц, хостелов и санаторно-курортных организаций за 2018–2023 годы.

инструменты - Python pandas, matplotlib.pyplot 

результат - построены распределения показателей, проверена гипотеза о росте показателей в 2023 году по сравнению с 2022 годом
